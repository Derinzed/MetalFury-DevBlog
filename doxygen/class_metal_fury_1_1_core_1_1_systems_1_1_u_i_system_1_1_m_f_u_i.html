const std::string & parentElementId )
inline

Insert a template into the currently active graph as a child of the element specified by parentElementId.

This operation validates that there are no identifier collisions between the template element ids and the existing active graph element ids. If any conflicts are found the insertion is rejected.

Parameters
templateIdId of the template to insert.
parentElementIdId of the parent element inside the active graph.
Returns
true on success, false on error (no active graph, missing template, unknown parent, or id conflicts).
