Detailed Description

Declarative UI graph builder and runtime storage for MetalFury UI.

This header defines a small, engine-agnostic representation of a UI hierarchy (UIGraph, UIElement, UIProperty) along with a fluent UIBuilder for constructing graphs declaratively. It also provides an MFUI adapter wrapper that stores multiple UIGraph instances, manages an "active" graph for rendering/interaction, and supports loading and instancing reusable templates into active graphs.

Concepts:

  • UIProperty: variant container of common property types (string, int, float, bool, vector).
  • UIElement: map of named properties and arbitrary children.
  • UIGraph: collection of root UIElements with a unique id.
  • UIBuilder: fluent builder that produces UIGraph instances.
  • MFUI: runtime container that stores graphs/templates and performs graph-level operations such as insertion, mutation and activation.
