Detailed Description

Declarative UI builder that constructs a UI graph.

UIBuilder provides a fluent API to declare UI elements and their hierarchical relationships. The builder produces a UIGraph via GetGraph() which can be loaded into an MFUI rendering/runtime implementation.

Example: UIBuilder b; b.Window("win", "Title", 800, 600, true, true) .Button("ok", "OK") .End(); auto graph = b.GetGraph();
